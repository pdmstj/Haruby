# 🌟 Haruby

**Node.js 기반 운세 추천 봇**

---

**Haruby**는 ‘하루’와 ‘행운’을 담은 작은 추천 봇이에요.  
매일매일, “오늘 뭐하지?” 싶을 때 Haruby가  
🎡 오늘의 운세, 🍜 음식, 🎧 음악, 🎮 게임을 가볍게 추천해줘요!

---

## 🚀 현재 제공 기능

### 📌 `GET /today-luck`
- 오늘의 **운세 / 음식 / 음악 / 게임**을 한 번에 추천해요.

### 📌 `GET /today-luck/:category`
- `foods`, `musics`, `games` 중 원하는 항목만 따로 받을 수 있어요.

### 📌 `GET /today-luck/birthday/fortune?birth=YYYY-MM-DD`
- **생년월일 기반** 운세 추천 + **별자리**까지 알려줘요!  
- (예: `birth=2006-08-19` → "사자자리", 8월 전용 운세 중 랜덤 1개)
